# Landing page đặt lịch Liên Quân

Trang landing page tĩnh để đặt lịch chơi Liên Quân cùng Mít và Táo.

## Run

Open [index.html](/Users/namquoc/Documents/New%20project/index.html) directly in your browser, or serve the folder locally:

```bash
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).

## Nội dung

- Hero giới thiệu đặt lịch chơi Liên Quân cùng Mít và Táo
- Ba gói: `Gói một tiếng`, `Gói ba tiếng`, `Gói một ngày`
- Khu vực cách đặt lịch
- Form tạo tin nhắn đặt lịch để gửi qua kênh liên hệ mong muốn

## Gợi ý cần bổ sung

- Giá từng gói
- Link Discord, Zalo, Messenger hoặc fanpage
- Lịch rảnh cố định của Mít và Táo
